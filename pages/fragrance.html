Fragrance Finder

Fragrance Finder is a work-in-progress web app that helps users discover perfumes similar to ones they already like. It uses the FragranceFinder API from RapidAPI to retrieve detailed scent metadata and similarity data.

The app compares perfumes using two main strategies: the built-in similarity index from the API, and shared fragrance notes like citrus, musk, vanilla, and oud. Users can search by name and explore suggestions based on shared scent profiles or recommendations from the API’s similarity algorithm.

I'm building this to learn more about working with third-party APIs, managing dynamic UIs in JavaScript, and structuring async search results. It's also a fun mix of tech and personal interest—perfect for perfume lovers or anyone looking for their next signature scent.
